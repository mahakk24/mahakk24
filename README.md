<h2>🚀 Featured Projects</h2>

<h3><a href="https://github.com/mahakk24/autoledger">⚡ AutoLedger</a> — Real-time Financial Intelligence for Startups</h3>

> Automates bookkeeping by classifying transactions, detecting anomalies, and forecasting cash flow — surfaced through a live React dashboard.

| Module | What it does |
|--------|-------------|
| 🧾 Transaction ingestion | Async FastAPI + PostgreSQL pipeline with CSV parsing |
| 🤖 ML classifier | Scikit-learn + TF-IDF categorises transactions automatically |
| 🚨 Anomaly detection | Isolation Forest flags unusual spending in real time |
| 📈 Cash flow forecast | Prophet / ARIMA time-series predictions |
| ⚡ Live dashboard | React + Recharts + WebSocket feed, Dockerised with CI/CD |

**`Python`** · **`FastAPI`** · **`PostgreSQL`** · **`Scikit-learn`** · **`Prophet`** · **`Celery`** · **`Redis`** · **`React`** · **`TypeScript`** · **`Docker`**
